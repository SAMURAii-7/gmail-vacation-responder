# OpenInApp Challenge

A Node.js based Gmail Vacation Responder

## How to use

Navigate to [http://localhost:3000/email](http://localhost:3000/email) to use the app.

## Installation

```bash
npm install
```

OR

```bash
yarn install
```

## Run (Normal Mode)

```bash
npm run start
```

OR

```bash
yarn run start
```

## Run (Dev Mode)

```bash
npm run dev
```

OR

```bash
yarn run dev
```
